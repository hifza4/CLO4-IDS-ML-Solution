# 🛡️ AI-Powered Intrusion Detection System 

##  Project Title
AI-Based Network Intrusion Detection System using Machine Learning



  ## Objective
The objective of this project is to develop an AI-powered Intrusion Detection System (IDS) that can detect malicious and normal network activities using machine learning techniques. The system helps improve cybersecurity by identifying attacks in network traffic data.



## Dataset Used
This project uses the **Network Intrusion Dataset** from Kaggle:

- Dataset Source: chethuhn/network-intrusion-dataset
- Loaded using: kagglehub

### Dataset Description:
- Contains multiple CSV files representing different network traffic scenarios
- Includes features such as:
  - Flow Duration
  - Packet Lengths
  - Flags
  - Bytes transferred



##  Technologies Used
- Python 
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- KaggleHub
- Jupyter Notebook



 Machine Learning Model
- Algorithm Used: **Random Forest Classifier**
- Data Preprocessing:
  - Missing value handling
  - Infinite value removal
  - Label Encoding
  - One-hot encoding
  - Feature scaling (StandardScaler)
